In mathematics, there are six fundamental operations used to manipulate sets. These operations allow us to perform various set operations and combine or modify sets. Let's discuss each operation with examples:

  1. Union:

    The union of two sets A and B, denoted by A ∪ B, is the set that contains all the elements that are in either set A or set B, or in both.

    Example:

    A = {1, 2, 3}

    B = {3, 4, 5}

    A ∪ B = {1, 2, 3, 4, 5}

  2. Intersection:

    The intersection of two sets A and B, denoted by A ∩ B, is the set that contains all the elements that are common to both set A and set B.

    Example:

    A = {1, 2, 3}

    B = {3, 4, 5}

    A ∩ B = {3}

  3. Set Difference:

    The set difference of two sets A and B, denoted by A - B or A \ B, is the set that contains all the elements that are in set A but not in set B.

    Example:

    A = {1, 2, 3, 4}

    B = {3, 4, 5}

    A - B = {1, 2}

  4. Complement:

    The complement of a set A, denoted by A', is the set of all elements in the universal set that are not in set A.

    Example:

    Universal set U = {1, 2, 3, 4, 5}

    A = {3, 4}

    A' = {1, 2, 5}

  5. Cartesian Product:

    The Cartesian product of two sets A and B, denoted by A × B, is the set of all ordered pairs (a, b) where a is from set A and b is from set B.

    Example:

    A = {1, 2}

    B = {3, 4}

    A × B = {(1, 3), (1, 4), (2, 3), (2, 4)}

  6. Power Set:

    The power set of a set A, denoted by P(A), is the set of all possible subsets of A, including the empty set and the set itself.

    Example:

    A = {1, 2}

    P(A) = {∅, {1}, {2}, {1, 2}}

These six operations allow us to perform various operations on sets and manipulate their elements. They are fundamental tools in set theory and have applications in many areas of mathematics and beyond.
